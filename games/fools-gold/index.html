Fool's Gold

“Unlock the power of an enchanted pickaxe, use it to climb, teleport and fight through a treacherous cave in an attempt to get back out”

Our third project at The Game Assembly was supposed to be a platformer. Fool’s Gold was created over the span of 12 weeks using the school’s own custom game engine called The Game Engine or TGE.



My Contributions

  • Implementation of player movement

  • Implementation of enemy AI a state machine

  • Input system with support for gamepad and mouse & keyboard inputs

  • Gameobject system

  • Animation implementation for player and enemies

  • Animation Manager for player
